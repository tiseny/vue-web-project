# vue-web-project

源链接：vue-spa-project (https://github.com/xiaoping6688/vue-spa-project)

## 使用

**创建项目**

```bash
$ git clone https://github.com/tiseny/vue-web-project.git
$ cd vue-web-project
```

**运行项目**

* 本地开发运行

```bash
$ npm run dev
```

* 构建

```bash
$ npm run build
```

* 运行构建

```bash
$ npm run prod
```
